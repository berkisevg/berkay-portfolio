# Berkay Sevgili — Portfolio

Statische Landingpage für Berkay Sevgili (Automation Engineer, Zürich).
Mobile-first, ein `index.html` plus `assets/`, ohne Build-Schritt.

**Live:** https://pascal-kuptz.github.io/berkay-portfolio/

## Lokal ansehen

Beliebiger Static-Server im Projektordner, zum Beispiel:

```bash
python -m http.server 8453
```

Dann http://localhost:8453 öffnen.

## Aufbau

- `index.html` — komplette Seite (HTML, CSS mit Design-Token-System, etwas JS für Scroll-Reveals)
- `assets/` — komprimierte Fotos (Porträt, Zürich)
- Schriften: Archivo, Instrument Serif, IBM Plex Mono (Google Fonts)
